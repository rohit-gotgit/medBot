## Overview

MedBot uses predefined **if/then logic** and keyword-based rules to simulate a medical assistant.  
It follows a structured conversational flow that allows users to interact through guided menus and question-answer sequences.

The system is designed to be:
- Always available  
- Easy to use  
- Capable of handling multiple users simultaneously  

---

## Features

- Rule-based medical query handling  
- Keyword-driven conversation flow  
- Symptom-based guidance  
- Menu-driven user interaction  
- Continuous 24/7 availability  
- Lightweight and fast response system

## Purpose of the Project

MedBot is a **prototype version of HealthMate** created to demonstrate how rule-based chatbots can be used in healthcare.  
It focuses on:
- Providing basic medical assistance  
- Guiding users through predefined healthcare flows  
- Offering structured information instead of free-form diagnosis  

This project is intended for educational and demonstration purposes.

---

## Technology Approach

The chatbot is built using:
- Rule-based decision logic  
- Keyword detection  
- Predefined response trees  
- Simple conversational routing  

This makes MedBot reliable, predictable, and easy to extend for future AI-based versions like HealthMate.

---

## Disclaimer

MedBot does not replace professional medical advice.  
It is intended only to provide general guidance and basic health information.  
Users should always consult a qualified healthcare professional for diagnosis and treatment.
